# PROJECT.md — project-starter-website

## One-liner
Personal reference site for the project-starter bootstrapping framework.

## Type
Static marketing/docs site. No auth, no database, no payments.

## Target User
The creator (solo operator who built and uses project-starter).

## Stack
- **Framework:** Next.js (App Router, SSG)
- **Hosting:** Vercel Hobby (free)
- **Styling:** Tailwind CSS
- **Animation:** GSAP 3.x + ScrollTrigger
- **Smooth scroll:** Lenis
- **Fonts:** TBD (Swiss + kinetic direction)

## Design Direction
Kinetic type meets Swiss International. Maximum typographic contrast, animated text reveals, strict grid with intentional breaks. Monochrome dominant with one sharp accent.

## Pages
Single page with anchored sections:
1. **Hero** — What is project-starter, one sentence + kinetic type animation
2. **Phase Flow** — The 6 phases visualized as a sequence
3. **Modules** — Categorized index (stack, business, planning)
4. **Commands** — Slash command quick reference
5. **Agents** — The 4 autonomous agent roles

## Not Building
- Auth, database, payments, analytics, email, AI features
- Multi-page routing
- Autonomous agents (disabled for this project)

## Cost
$0/month (Vercel Hobby)
